
## Описание

Автоматизированное тестирование 3х сценариев для https://sbis.ru/ .
Реализация на Python3 и Selenium Webdriver, реализован паттерн PageObject.

## Инструкция по запуску

Установка пакета

```
poetry install
```

Сборка пакета

```
poetry build
```

Запуск тестов

```
pytest
```
или

```
python3 -m pytest
```
